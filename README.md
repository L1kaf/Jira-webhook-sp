Данный вебхук предназначен для автоматического подсчета SP в задачах на основе подзадач. Работает через Docker контейнер. В jira необходимо в URL указать адрес хука с issue=${issue.key} (получаем номер задачи, от которой прилетел хук). пример: 
```
http://адрес:порт/webhook?issue=${issue.key}
```
В файле .env прописывается путь до логов, адрес jira и токен УЗ, от которой будут происходит переходы.
